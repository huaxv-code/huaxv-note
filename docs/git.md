# git 使用

## 克隆

```
git clone <url>
```

## 初始

```
git init
```

## 状态

```
git status
```

## 添加

```
git add .

git add <file>
```

## 改名

```
git mv <old> <new>
```

## 删除

```
git rm <file>
```

## 添加后移出

```
git rm --cached <file>
```

## 移入本地仓库

```
git commit -m "message"
```
