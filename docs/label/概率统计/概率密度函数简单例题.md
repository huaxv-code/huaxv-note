# 概率密度函数简单例题

!!! question

    设随机变量 $X$ 的概率密度函数为：

    > $0 \leq x < 1 时，f(x) = x$
    > 
    > $1 \leq x < 2 时，f(x) = 2 - x$
    > 
    > 其他 $f(x) = 0$

    求 $X$ 的分布函数。

分布函数与概率密度函数的关系：
> $f(x) = {F}'(x)$

所以 $F(x) = \int_{-\infty}^xf(x)dx$

当 $x < 0$ 时：
> $F(x) = 0$

当 $0 \leq x < 1$ 时：
> $F(x) = \int_{0}^{x}xdx = \frac{1}{2}x^2$

当 $1 \leq x < 2$ 时：
> $F(x) = \int_{0}^{1} x dx + \int _ {1}^{x} 2 - x dx = -\frac{1}{2}x ^ 2 + 2x - 1$

当 $x \geq 2$ 时：
> $F(x) = 1$
