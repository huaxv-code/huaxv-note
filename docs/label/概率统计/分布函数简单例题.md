# 分布函数简单例题

!!! question

    设连续型随机变量 $X$ 的分布函数为：
    > $F(x) = A + Be^{-\lambda x}，x \geq 0, \lambda > 0$

    求出 $A、B$ 的值；

    求出概率密度函数 $f(x)$。

由于当 $x \rightarrow +\infty$ 时，$F(x) \rightarrow 1$，所以：
> $A = 1$

由于该分布函数连续，所以 $F(0) \rightarrow F(^-0) = 0$，所以：
> $A + B = 0$ 解得 $B = -1$

所以概率密度函数 $f(x) = {F}'(x)$:
> $f(x) = \lambda e ^ {- \lambda x}$
